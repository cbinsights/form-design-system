fds-icons

React components are generated for all icons shown below.

Click on an import statement to copy it.

All icon components accept the following React props
Prop Type Default Description
color String COLOR_CHARCOAL Color of fill for SVG
size one of: 'xs', 's', 'm', 'l', 'xl' s Standard media size of icon
customSize Number - Sets a non-standard size for the icon. Prefer using the size prop.
