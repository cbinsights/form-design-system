Spacing Helpers

Flush

Removes spacing from the given side of an element. Sets both padding and margin to 0;

class description
flush--top Removes top spacing from element
flush--right Removes right spacing from element
flush--bottom Removes bottom spacing from element
flush--left Removes left spacing from element
flush--all Removes spacing from all sides of an element

Margin

Adds margin to the given side of an element.

class description
margin--top Adds standard margin to top of element
margin--right Adds standard margin to right of element
margin--bottom Adds standard margin to bottom of element
margin--left Adds standard margin to left of element
margin--all Adds standard margin to all sides

Padding

Adds padding to the given side of an element.

class description
padding--top Adds standard padding to top of element
padding--right Adds standard padding to right of element
padding--bottom Adds standard padding to bottom of element
padding--left Adds standard padding to left of element
padding--all Adds standard padding to all sides

Position Helpers

Algin

eets text-align.

class description
align--left Sets text-align to left
align--center Sets text-align to center
align--right Sets text-align to right

Vertical Align

Sets vertical-align property. Note that vertical-align only applies to elements displayed inline, inline-block, table-cell or inline-flex.

class description
valign--top Sets vertical-align to top
valign--center Sets vertical-align to middle
valign--bottom Sets vertical-align to bottom

Child Align

Positions direct child(s) of element using flex properties. These classes modify flex properties. Avoid combining alignChild classes with flexbox layout elements.

Classes follow this signature:

.alignChild--(left|right|center)--(top|bottom|center)

<div
  class="alignChild--center--center"
  style="border: 1px dotted red; height: 100px; width: 100px;"
>
  <div style="background: pink;">child</div>
</div>

Typesetting

Heading Tags

By default, HTML heading tags are set to match the default body text. To set the display of a given heading tag, use one of the typesetting classes below.

<h1>Heading1</h1>
<h2>Heading2</h2>
<h3>Heading3</h3>
<h4>Heading4</h4>
<h5>Heading5</h5>
<h6>Heading6</h6>

Heading 1

<p class="typeset--head1">This is a top-level heading</p>

Heading 2

<p class="typeset--head2">This is a secondary heading</p>

Heading 3

<p class="typeset--head3">This is a tertiary heading</p>

Heading 4

<p class="typeset--head4">This is a minor heading</p>

Body

Default body text. No classes necessary.

<p>This is default body text</p>

Data

Typsetting for data (e.g. tables).

<p class="typeset--data">This is data text. Good for tables.</p>

Caption

Tiny text for captions and legalese. This is not very readable; only use in special circumstances.

<p class="typeset--caption">This is a caption. &copy; 2018</p>

Modifies weight of text.

<p class="typemod--book">Book Text</p>

Text flow

Truncate text

Helpers to adjust text wrapping with CSS.

Allows compontents to render full strings while CSS does the job of clamping text to one line.

Class Description
.wrap--singleLine forces text to a single line (nowrap)
.wrap--singleLine--truncate forces text to a single line (nowrap) and truncates with "…" entity

Text wrap control

Helpers to adjust text wrapping with CSS.

Allows compontents to render full strings while CSS does the job of clamping text to one line.

Class Description
.wrap--hyphenate Allows breaking on word with hyphenation
