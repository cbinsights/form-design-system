Spacing Helpers

Flush

Removes spacing from the given side of an element. Sets both padding and margin to 0;

class description
flush--top Removes top spacing from element
flush--right Removes right spacing from element
flush--bottom Removes bottom spacing from element
flush--left Removes left spacing from element
flush--all Removes spacing from all sides of an element

Position Helpers

Algin

Sets text-align with left/right fallbacks for IE11.

class description
align--start Sets text-align to left/start
align--center Sets text-align to center
align--end Sets text-align to right/end

Vertical Align

Sets vertical-align property. Note that vertical-align **only applies to elements displayed inline, inline-block, table-cell or inline-flex.

class description
valign--top Sets vertical-align to top
valign--center Sets vertical-align to middle
valign--bottom Sets vertical-align to bottom

Child Align

Positions direct child(s) of element using flex properties. These classes modify flex properties. Avoid combining alignChild classes with flexbox layout elements.

Classes follow this signature:

.alignChild--(left|right|center)--(top|bottom|center)

<div
  class="alignChild--center--center"
  style="border: 1px dotted red; height: 100px; width: 100px;"
>
  <div style="background: pink;">child</div>
</div>
