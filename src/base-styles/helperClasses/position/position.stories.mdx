## Child Align
Positions direct child(s) of element using flex properties.
**These classes modify `flex` properties.**
Avoid combining `alignChild` classes with flexbox layout elements.

<Story name="Child Align">
  <>
    <TableLayout
      isCSS
      headers={['class', 'description', 'sample']}
      rows={[
        ['alignChild--center--center', 'Adds standard margin to all sides'],
        ['alignChild--center--top', 'Adds standard margin to all sides'],
        ['alignChild--center--bottom', 'Adds standard margin to all sides'],
        ['alignChild--left--center', 'Adds standard margin to bottom of element'],
        ['alignChild--left--top', 'Adds standard margin to top of element'],
        ['alignChild--left--bottom', 'Adds standard margin to right of of element'],
        ['alignChild--right--center', 'Adds standard margin to all sides'],
        ['alignChild--right--top', 'Adds standard margin to left of element'],
        ['alignChild--right--bottom', 'Adds standard margin to all sides'],
      ].map(row => [
        row[0],
        row[1],
        { 
          style: { paddingTop: 0, paddingBottom: 0 },
          children: <div className={row[0]} style={positionStyles}><div style={boxStyles}/></div>
        }
      ])} 
    />
  </>
</Story>

## Vertical Align
Sets `vertical-align` property. **Only applies to `inline`, `inline-block`,
`table-cell` or `inline-flex` elements**.

<Story name="Vertical Align">
  <TableLayout
    isCSS
    headers={['class', 'description', 'sample']}
    rows={[
      ['valign--top', 'Sets `vertical-align` to `top`'],
      ['valign--center', 'Sets `vertical-align` to `middle`'],
      ['valign--bottom', 'Sets `vertical-align` to `bottom`'],
    ].map(row => [
      row[0],
      row[1],
      { 
        style: { backgroundColor: FDS.COLOR_HAZE, position: "relative" }, 
        children: (
          <>
            <span
              className={`${row[0]} display--inlineBlock`}

            >
              CB
            </span>

          </>
        )
      }
    ])}

</Story>
